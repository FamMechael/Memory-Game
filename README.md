# Memory Game

A simple web-based Memory Game built using vanilla JavaScript, HTML, and CSS. This project was created as a practical exercise to practice DOM manipulation and browser data persistence.

## Features

*   **Custom Grid:** Allows users to input grid dimensions (rows and columns) before starting.
*   **Timer & Score Tracking:** Keeps track of the remaining time and the user's score during the game.
*   **Data Persistence:** Uses `localStorage` to save game history, allowing past match results to display even after refreshing the page.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript (ES6+)

## How to Run

1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.
3. Enter your name, select the grid size, set the timer, and click **Play**.

## What I Practiced in this Project

*   Handling browser events and managing game states.
*   Dynamically creating and updating DOM elements based on user interaction.
*   Storing, parsing, and retrieving arrays of objects using `localStorage`.
